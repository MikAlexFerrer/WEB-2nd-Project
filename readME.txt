Welcome to our project of Web Development. In this text we will give instructions and indications about how to open this website locally on your computer.
First, let's see if you have the following programs installed:
1. Visual Studio Code
  . NodeJS
2. MongoDB
  . Make sure Visual is linked to MongoDB

Next, on a new terminal created on Visual, run the following program:

"npm run devServer"

Finally, open a new tab and type:
"localhost:3000"

Enjoy our website and have a great day :D.
